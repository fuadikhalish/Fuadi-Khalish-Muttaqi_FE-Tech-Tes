HashMicro Front end Test
=====================================

CARA MENJALANKAN PROJECT:
-------------------------
Untuk menjalankan project ini sangat sederhana, hanya perlu mengakses file:
index.html

CATATAN PENTING:
- Project ini menggunakan CDN, sehingga membutuhkan koneksi internet
- Tidak perlu npm install atau setup server
- Semua dependency sudah di-load via CDN

STRUKTUR FILE:
- index.html         (File utama - buka file ini)
- employeeStore.js   (Pinia store untuk state management employee)
- styles.css         (Custom styling dan animasi)
- assets/            (Folder untuk gambar dan aset)

TEKNOLOGI YANG DIGUNAKAN:
- Vue 3 (via CDN)
- Pinia 2 (State Management via CDN)
- Vuetify 3.4.0 (for Material UI aesthetic)
- Bootstrap 5.3.8 (CSS Framework)

FITUR:
- CRUD Employee (Create, Read, Update, Delete dengan Pinia store)
- Modal confirmations untuk setiap aksi
- Toast notifications untuk feedback
- Loading states untuk better UX
- Form validation
- Responsive design
- Animated background shapes
